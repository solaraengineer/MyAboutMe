Django

What is Django?

Explanation

A high-level Python framework for rapid web development with clean, pragmatic design.

REST API

What is REST API?

Explanation

REST APIs allow systems to communicate over HTTP using methods like GET, POST, PUT, DELETE.

JWT

What is JWT Token?

Explanation

JWTs are tokens used for securely transmitting user info between client and server.

CSRF

What is CSRF Token?

Explanation

A CSRF token is used to protect against Cross-Site Request Forgery attacks.

Django Admin

What is Django Admin?

Explanation

A built-in admin panel to manage models and database entries without writing SQL.

ORM

What is Django ORM?

Explanation

Object-Relational Mapper lets you interact with the database using Python classes.

MVT

Django's Architecture?

Explanation

Django uses MVT: Model (DB), View (Logic), Template (Frontend) — similar to MVC.

Fun Fact

Where did Django start?

Fun Fact

It was created by developers at a newspaper to manage content-heavy websites fast.

Fun Fact

Why the name "Django"?

Fun Fact

It's named after Django Reinhardt — a legendary jazz guitarist.

Security

Is Django Secure?

Yes

Django has built-in protection against SQL injection, XSS, CSRF, and more.
